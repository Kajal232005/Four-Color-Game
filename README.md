# 🔴🟡 Connect Four Game using Python

A graphical **Connect Four Game** built using **Python, Pygame, and NumPy**. Two players take turns dropping colored discs into columns, and the first player to connect four pieces horizontally, vertically, or diagonally wins.

## 🎮 Features

* Interactive graphical interface using Pygame
* Two-player gameplay
* Mouse-based controls
* Win detection:

  * Horizontal
  * Vertical
  * Diagonal
* Real-time board updates
* Winning message display

## 🛠️ Technologies Used

* Python 3
* Pygame
* NumPy

## 🎯 How to Play

1. Launch the game.
2. Player 1 uses **Red** pieces.
3. Player 2 uses **Yellow** pieces.
4. Move the mouse over a column.
5. Click to drop a piece.
6. First player to connect **4 pieces** wins.

## 🧠 Game Logic

The game checks for:

* Horizontal matches
* Vertical matches
* Positive diagonal matches
* Negative diagonal matches
